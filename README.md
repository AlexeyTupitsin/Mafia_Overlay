# Графика трансляции спортивной мафии

Веб-приложение для трансляций: **панель ведущего** + **оверлей для OBS** (Browser Source).
Сервер хранит состояние игры и мгновенно синхронизирует панель и оверлей по WebSocket.

Работает одинаково на **Windows** и **macOS** (и на Linux) — всё, что нужно, это Node.js.

## Как скачать

Без установки Git — архивом:

1. Откройте страницу проекта: <https://github.com/AlexeyTupitsin/Mafia_Overlay>
2. Зелёная кнопка **`< > Code`** → **Download ZIP**
   (прямая ссылка: <https://github.com/AlexeyTupitsin/Mafia_Overlay/archive/refs/heads/main.zip>).
3. Распакуйте архив в любую папку.

> В архиве **нет** папки `node_modules/` и файла `config.json` — их добавлять не нужно:
> зависимости ставятся командой `npm install` (см. ниже), а `config.json` требуется только
> для интеграции с трекером рейтинга и для обычной работы оверлея не нужен.

## Что установить

1. **Node.js 20 LTS или новее** — единственная обязательная зависимость.
   Скачайте с официального сайта <https://nodejs.org/> и установите со всеми настройками
   по умолчанию.
   - *Windows:* запустите скачанный `.msi` и жмите «Далее».
   - *macOS:* запустите скачанный `.pkg`, либо через Homebrew: `brew install node`.

   Проверить установку (в терминале/командной строке):
   ```
   node --version
   ```
   Должна показаться версия `v20…` или выше.

2. **Google Chrome** — нужен **только** для дев-утилиты скриншотов (`tests/test-shot.mjs`).
   Для обычной работы трансляции не требуется.

## Установка проекта

В терминале перейдите в папку проекта и один раз выполните:
```
npm install
```
Команда скачает зависимости (Express, ws, multer, sharp) в `node_modules/`.

> **Как открыть терминал в папке проекта**
> - *Windows:* откройте папку в Проводнике, в адресной строке наберите `cmd` и нажмите Enter.
> - *macOS:* в Finder откройте папку, правый клик → «Новый терминал в папке».

## Запуск

Кросс-платформенный способ (Windows, macOS, Linux):
```
npm start
```

*Windows:* для удобства можно просто дважды кликнуть по `start.bat` — он сам запустит сервер.

После старта в консоли печатаются адреса. Откройте:
- **Панель ведущего:** <http://localhost:3000/control> — в обычном браузере;
- **Оверлей:** <http://localhost:3000/overlay> — добавляется в OBS (см. ниже).

Порт настраивается в `config.json` (поле `port`) или переменной окружения `PORT`.

### Панель с планшета

Планшет должен быть в той же сети, что и компьютер трансляции. На планшете откройте
`http://<IP-адрес-компьютера>:3000/control` — адрес печатается в консоли при старте сервера.

## Добавление оверлея в OBS

1. Источники → **+** → **Браузер** (Browser Source).
2. Настройки источника:
   - URL: `http://localhost:3000/overlay`
   - Ширина: **1920**, высота: **1080**, FPS: **30**
   - «Выключать источник, когда он не виден» (Shutdown source when not visible) — **выкл**
   - «Обновлять браузер, когда сцена становится активной» — **вкл**
3. Фон оверлея прозрачный — графика ляжет поверх видео.

## Как пользоваться

- **Подготовка:** введите название, ники (подсказки из справочника прошлых игр), загрузите
  фото (клик или drag-and-drop), назначьте роли (6 мирных, 2 мафии, дон, шериф) и нажмите
  «Начать игру». Кнопка «⇄» меняет игроков местами.
- **День:** тап «🎙 Слово» — речь игрока (подсветка + таймер на оверлее), кнопки 60/30 в шапке —
  запуск таймера; «☝ Выставить» — кандидатура на голосование.
- **Голосование:** «Начать голосование», у каждой кандидатуры укажите число голосов,
  «Завершить голосование» — итог рассчитывается автоматически (равенство → перестрелка →
  «поднять всех»).
- **Ночь:** кнопка «Ночь N →» в шапке; блоки «Отстрел», «Проверка дона», «Проверка шерифа».
- **ЛХ:** при первой смерти в первую ночь панель сама предложит ввести лучший ход.
- **Фолы:** «+/−» на карточке; на 3-м фоле пометка «без речи», на 4-м — предложение удалить.
- **Отмена:** «⎌ Отменить» откатывает последние действия (до 50 шагов).
- **Завершение:** меню ☰ → «Завершить игру» → результат. Протокол доступен из архива
  (☰ → «Архив игр») — печатная страница и JSON-экспорт.
- **Тумблеры оверлея:** меню ☰ — показать/скрыть плашки, трекер, бар голосования, таймер, роли.

## Импорт игроков из трекера рейтинга

Панель умеет подтягивать игроков (ник + аватар) из трекера **Iron Maf / MafiaClubTracker**.
По умолчанию интеграция выключена. Чтобы включить, впишите в `config.json` блок `tracker`
(шаблон — в `config.example.json`):

```json
"tracker": {
  "enabled": true,
  "supabaseUrl": "https://ВАШ-ПРОЕКТ.supabase.co",
  "serviceRoleKey": "service_role-ключ из Supabase трекера"
}
```

`service_role`-ключ берётся в панели Supabase трекера (Project Settings → API). Он остаётся
**только на сервере оверлея** и в браузер не передаётся, поэтому `config.json` добавлен в
`.gitignore` — не публикуйте его.

После перезапуска на экране «Подготовка игры» появится кнопка **«⬇ Импорт из рейтинга»**:
клик загружает игроков в справочник, и их ники сразу доступны при вводе состава (аватар
подставляется автоматически). Если блок `tracker` не настроен — кнопки нет, остальная работа
оверлея не меняется.

## Надёжность

- Состояние автосохраняется в `data/autosave.json` — перезапуск сервера или OBS ничего не теряет.
- Оверлей при обрыве связи тихо переподключается, не показывая ошибок в кадре.
- Завершённые игры складываются в `data/games/`, фото игроков — в `uploads/`.

## Тесты

Интеграционные проверки лежат в `tests/` и работают по WebSocket с **запущенным сервером**.
Сначала в одном терминале запустите `npm start`, затем в другом:
```
npm test                       # сквозной сценарий игры (tests/test-scenario.mjs)
node tests/test-scores.mjs     # проверка баллов и протокола
node tests/test-remarks.mjs    # замечания/фолы
```
`tests/test-shot.mjs` делает скриншот страницы через Chrome (путь к Chrome зашит под Windows) —
это дев-утилита для визуальной проверки, в обычной работе не нужна.

Юнит-тест импорта из трекера сервер не требует (сеть мокается): `node tests/test-tracker-import.mjs`.

Линтер: `npm run lint`.

## Структура проекта

```
server/   — Node.js-сервер (Express + ws): состояние, действия, автосейв, REST
public/   — статика: /control (панель), /overlay (оверлей), /shared (общие модули)
tests/    — интеграционные тесты по WebSocket
data/     — autosave.json, справочник игроков, архив игр (создаётся при работе)
uploads/  — фотографии игроков (создаётся при работе)
config.json — порт, значения по умолчанию и настройки трекера (не коммитится; шаблон config.example.json)
start.bat — двойной клик для запуска на Windows
```
